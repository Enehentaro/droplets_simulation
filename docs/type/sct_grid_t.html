type, public :: sct_grid_t

SC/TETRA メッシュクラス. 必要最低限の変数のみ保持. 変数名はフォーマットに準拠.
メッシュそのものを取り扱うのでメモリ圧迫する可能性大. ソルバ内で使う場合はサブルーチンのローカル変数として扱う方が無難(自動開放されるはず)
